## 👋 About Me

I'm a passionate **Web Developer** on a journey to becoming a **Full Stack Developer**. I specialize in creating beautiful, performant, and user-centric web applications while continuously expanding my skills in backend development and database management.

My approach combines **clean code**, **intuitive UI/UX**, and **robust architecture** to build complete solutions that users love to interact with. I'm driven by the challenge of understanding the full stack and building applications from the ground up.
